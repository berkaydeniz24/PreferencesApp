## Project Description
This project is a multi-screen React Native application developed for Lab 6.
It demonstrates real-world state management patterns using:

- Local state with `useState`
- Global state with Context API
- Side effects with `useEffect`
- Persistent storage using AsyncStorage
- Logic reuse with a custom hook

---

## Features
- Mock login / logout flow
- Persisted authentication state
- Global user context (no prop drilling)
- Light / Dark theme preference
- Theme persistence with AsyncStorage
- Settings screen for user preferences
